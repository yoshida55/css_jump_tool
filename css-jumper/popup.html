🎯 CSS Jumper

HTMLの要素を右クリック → CSSの該当行へ直接ジャンプ

履歴があれば📁で同じ親フォルダから別プロジェクトを選択可能

Live Serverで開いているページから自動検出

ONにするとページ読込時にFlexboxの情報を自動表示

使い方

  1. ①にプロジェクトのフルパスを入力
  2. ②でCSSファイルを選択（Ctrl+クリックで複数選択）
  3. Webページで要素を右クリック
  4. 「VS CodeでCSS開く」を選択
  5. → 既存のVS Codeで該当行に自動ジャンプ！
