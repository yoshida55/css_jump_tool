# CSS Jumper セットアップ手順書

**HTML 要素を右クリック → VS Code で該当 CSS の行に直接ジャンプする拡張機能**

---

## 📋 インストール手順

### 1. Chrome 拡張機能のインストール

1. Chrome で `chrome://extensions/` を開く
2. 「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `css-jumper` フォルダを選択

---

### 2. vscode プロトコルの登録（重要！）

**PowerShell（管理者として実行）**で以下を実行：

```powershell
# VS Codeのパスを自動検出
$vscodePath = "C:\Users\$env:USERNAME\AppData\Local\Programs\Microsoft VS Code\Code.exe"

# パスの確認
if (Test-Path $vscodePath) {
    Write-Host "✓ VS Code発見: $vscodePath" -ForegroundColor Green
} else {
    Write-Host "✗ VS Codeが見つかりません" -ForegroundColor Red
    exit
}

# レジストリに登録
New-Item -Path "HKCU:\Software\Classes\vscode" -Force | Out-Null
Set-ItemProperty -Path "HKCU:\Software\Classes\vscode" -Name "(Default)" -Value "URL:vscode"
Set-ItemProperty -Path "HKCU:\Software\Classes\vscode" -Name "URL Protocol" -Value ""
New-Item -Path "HKCU:\Software\Classes\vscode\shell\open\command" -Force | Out-Null
Set-ItemProperty -Path "HKCU:\Software\Classes\vscode\shell\open\command" -Name "(Default)" -Value "`"$vscodePath`" --open-url `"%1`""

Write-Host "✓ vscodeプロトコルの登録完了！" -ForegroundColor Green
```

**動作確認**:

1. Chrome を再起動
2. アドレスバーに `vscode://file/C:/` と入力して Enter
3. VS Code が開けば OK！

---

### 3. 初回設定

1. 拡張機能アイコンをクリック
2. **① プロジェクトパス**を入力（例: `D:\02_作業\...\01_step_05`）
3. **②CSS ファイルを選択**ボタンをクリック
4. Ctrl キーを押しながら CSS ファイルを全選択
5. 「✓ ○ 個の CSS ファイルを読み込みました」と表示されれば OK

---

## 🚀 使い方

### 方法 1: 右クリックメニュー

1. Live Server で HTML を開く
2. 要素を右クリック → 「VS Code で CSS 開く」
3. VS Code で該当行が開く！

### 方法 2: ショートカット（推奨）

1. **Alt + クリック** で要素をクリック
2. 右クリックメニューなしで直接 VS Code にジャンプ！

### 方法 3: サイズ表示

1. 拡張機能アイコンをクリック
2. 画面幅を選択（1280px / 375px / カスタム）
3. 「表示開始」をクリック
4. 全要素のサイズがオーバーレイ表示される

## ⚠️ トラブルシューティング

### VS Code が開かない

**解決策**: レジストリを再登録

```powershell
Set-ItemProperty -Path "HKCU:\Software\Classes\vscode\shell\open\command" -Name "(Default)" -Value "`"C:\Users\$env:USERNAME\AppData\Local\Programs\Microsoft VS Code\Code.exe`" --open-url `"%1`""
```

実行後、Chrome 再起動

### 「CSS ファイルを読み込んでください」と表示される

**解決策**: 拡張機能アイコン → CSS ファイルを再選択

### ページをリロードしたら動かない

**解決策**: F5 でページ再読み込み

### クラス名が見つからない

**確認事項**:

- 除外ファイル（reset.css, normalize.css, sanitize.css）はスキップされます
- CSS ファイルを再読み込みしてみてください
- クラス名のスペルを確認してください

## 📁 対応ファイル構造

CSS Jumper は以下のような構造に対応しています：

```
プロジェクト/
├── css/
│   ├── style.css      ← 検索対象
│   ├── main.css       ← 検索対象
│   └── reset.css      ← 除外
├── index.html
└── ...
```

**注意**: CSS ファイルは任意のフォルダに配置可能です。
ファイル選択時の相対パスが保存されるため、`/css/` フォルダ以外でも動作します。
