## 🎯 機能別一覧

### CSSジャンプ関連（ブラウザ）

| キー | 機能 | 詳細 |
|------|------|------|
| `Alt+クリック` | 通常のCSSジャンプ | CSSファイルのみ開く |

## 🧠 覚え方のコツ

### パターン1: ブラウザ操作（Alt系）
```
Alt+クリック    → CSSジャンプ
Alt+F          → Flexトグル
Ctrl+ダブルクリック → モバイルCSS
```

### パターン2: VS Code検索（Ctrl+Shift系）
```
Ctrl+Shift+H → HTML検索
Ctrl+Shift+A → セクション一覧
Ctrl+Shift+M → メモ検索
```

### パターン3: VS Code移動（Ctrl+矢印系）
```
Ctrl+↓ → 次のセクション
Ctrl+↑ → 前のセクション
```

### パターン4: AI関連（Ctrl系）
```
Ctrl+I          → Claude質問（Iは Input の I）
Ctrl+Shift+/    → Copilot解説（/はコメント記号）
```

## ⚠ 注意事項

### Chrome拡張のキー制限
- `Ctrl+Alt+[key]` はChromeで使用禁止（システム予約）
- `Alt+F`は使用可能（`Alt+Shift+F`から変更済み）

### VS Codeのキー競合
- `Ctrl+Alt+V` は Paste Image拡張が使用中 → SVG挿入は`Ctrl+Alt+S`

### DevToolsレスポンシブモード
- ダブルクリックは動作しない（Chrome仕様でズーム機能が優先）
- 代替: **Flexラベルクリック** または **Alt+クリック**
